fluid.multiarray

Multichannel arrays

Description

An abstraction that assists in the management and creation of multichannel arrays.

Discussion

This abstraction allows the creation of multichannel arrays. Note that it requires invoking with the clone object to work.

fluid.multiarray is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Arguments

channels [int]

Optional
The number of channels for the multiarray

name [symbol]

Optional
The name of the multiarray

size [int]

Optional
The number of elements that the array should store
