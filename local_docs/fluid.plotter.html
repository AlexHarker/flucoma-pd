fluid.plotter

Plot data in a two-dimensional graph.

Description

A GUI object that plots data in a two-dimensional graph.

Discussion

The fluid.plotter object can be used as a versatile solution for plotting points in a two-dimensional graph. It is designed to easily accept the contents of a fluid.dataset via a dictionary as coordinates within the graph as well as the contents of a fluid.labelset~ via a dictionary as colours for points. There is also an 'advanced' interface that allows the user to specify the coordinates and colours of the points in a more flexible and bespoke manner. See the help file for more information on this.

fluid.plotter is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Arguments

Attributes

axis [int] (default: 0)

In what direction to copy data from the source. 0 copies frame by frame, 1 copies channel-by-channel.

source [symbol] (default: )

Sets the source buffer~ to copy data from.

startchan [int] (default: 0)

For multichannel input buffers, which channel to begin copying from. The default is 0.

Messages

Set the x-axis range.

    The maximum value for the x-axis

Set the y-axis range.

Argument details:

    The minimum value for the y-axis

  • max:

    The maximum value for the y-axis

range

Arguments: min (number) max (number)

Set the x- and y-axis ranges.

Argument details:
  • min:

    The minimum value for the x-axis and y-axis

  • max:

    The maximum value for the x-axis and y-axis

highlight

Arguments: identifiers (list)

Emphasises the given identifier's points in the plot

Argument details:
  • identifiers:

    Any number of identifiers

clear

Arguments:

Clears the graph and all associated data and settings.

Argument details:

setlabels

Arguments: text (symbol)

Set the labels for each point according to their labels in a fluid.labelset.

Argument details:
  • text:

    The name of a text object containing data from a fluid.dataset. This message must enter the left inlet.

setpoints

Arguments: text (symbol)

Set the position for points by their data in a fluid.dataset.

Argument details:
  • text:

    The name of a text object containing labels from a fluid.labelset. This message must enter the right inlet.
